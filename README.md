# sector-wise-fundamental-screener
# Overview
This project builds a structured scoring model to evaluate and rank companies within sectors based on financial performance.

# Objective
To perform sector-wise fundamental analysis and identify top-performing stocks using a weighted scoring approach.

# Methodology
- Sector segmentation: Banking, Automobile, Consumer Durables
- Financial ratios used:
  - ROE, ROCE (Profitability)
  - P/E (Valuation)
  - Debt/Equity, ICR (Risk)
  - Sales & Profit Growth (Growth)
- Weighted scoring model to rank companies

# Key Insights
- The banking sector is structurally leveraged
- Consumer Durables show low financial risk
- The automobile sector shows cyclical recovery trends

# Tools Used
- Microsoft Excel
